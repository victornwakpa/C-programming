HOW TO CREATE AND EXECUTE C FILES ON WINDOWS

1. Copy the file path of the c file. example: C:\Users\PCname\Documents\C\C-programming

2. Go to the terminal and type: cd <file-path>

N/B: you must have a gcc compile for the next step
type: gcc --version
to check the verion you have. If it returns error or command not found.
Then go and install gcc compile frst

3. Type: gcc <file-name>
example: gcc HelloWorld.c

There will be a new file called a.exe 

4. Type: ./<filename>
to execute the task
example ./a
